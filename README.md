# Beef Raylib 3D Game

## Overview
This project is a game POC developed using BeefLang and using the raylib graphics library.

Supported platforms are currently Win64 and Wasm32 (without shadows currently).

### Implemented:
* Simple camera movement
* Cascade Shadow Maps
